birlashtirib natijani qaytaradi

const arr1 = [1, 2,]
const arr1 = [3, 4, 5]
const result = arr1.concat(arr2);

indexof()

array'ni ichidan elementni izlab uning
joylashgan o'rnini (index) qaytaradi

const array = [10, 20, 30]
console.log( array.indexOf(30) );
Natija : 2
forEach()

har bir element uchun berilgan funksiyani chaqiradi.

const newNumbers [2, 3, 4, 5]
nummbers.forEach((num) => {
newNumbers.push(num ** 2)
} )
console.log(newNumbers);
Natija: [4, 9, 16, 25]
map()

har bir massiv elementi uchun
berilgan funksiyani chaqirib, yangi array yaratadi.

const newNumbers [2, 3, 4, 5]
const mapNumbers = number.map((num) => num ** 2 )
console.log(mapNumbers);
Natija: [4, 9, 16, 25]
filter()

berilgan shartni bajargan elementlarni
saralab olib, yangi massiv yaratadi.

const number = [1, -1, 2, 3]
const filter = numbers.filter(n => n => 0);
console.log(filtered);
Natija : [1, 2, 3]
includes()

massivda ko'rsatilgan element mavjudligini tekshiradi (true, false).

const array = [10, 20, 30]
console.log( array.includes(20) );
Natija : true
slice()

massivni ko'rsatilgan qismini tanlab, qaytaradi.

const array = [10, 20, 30]
console.log( array.slice(0, 2));
Natija : [10, 20]
push()

push yordamida array oxiriga so'z qo'shish mumkin.

let languages = ['js', 'java]
languages.push('go') console.log(languages );
Natija : ['js', 'java', 'go']
unshift()

unshift yordamida array boshiga so'z qo'shish mumkin.

let languages = ['js', 'java']
languages.unshift('go')
console.log(languages );
Natija : ['go', 'js', 'java']
pop()

pop metodidan foydalanib array ichidagi oxirgi
elementni o'chirish mumkin va boshqa qo'shish mumkin.

const languages = ['js', 'java', 'go', 'swift']
const deletedLanguage = languages.pop();
console.log(languages );
Natija : ['js', 'java', 'go'] / X : swift
shift()

shift metodidan foydalanib array ichidagi boshidagi
elementni o'chirish mumkin va boshqa qo'shish mumkin.

const languages = ['js', 'java', 'go', 'swift']
const deletedLanguage = languages.shift();
console.log(languages );
Natija : ['swift', 'java', 'go'] / X : js
some()

Usul someba'zi massiv qiymatlari sinovdan
o'tganligini tekshiradi. U trueyoki qaytaradi false.

let numbers = [4, 6, 14, 16]
let isSomeGreaterThan6 = numbers.some(n => n > 6) // true
let isSomeLessThan4 = numbers.some(n => n < 4)
// false
every()

Usul someba'zi massiv qiymatlari sinovdan
o'tganligini tekshiradi. U trueyoki qaytaradi false.

let numbers = [4, 6, 14, 16]
let isEverythingGreaterThan6 = numbers.every(n => n > 6) // false
let isEverythingLessThan20 = numbers.some(n => n < 20) // true

reduce()

Qayta qo'ng'iroq qilish funktsiyasi
reduceto'rtta argumentni olishi mumkin:

let numbers = [1, 2, 3, 4]
let total = numbers.reduce((total, currentNum) => total + currentNum)
map()

Funktsiyaga o'tkazilgan qayta qo'ng'iroq
map()uchta argumentdan birini qabul qilishi mumkin:

let numbers = [1, 2, 3, 4]
let doubledNumbers = numbers.map(n => n * 2)
let doubledOddIndexNumbers = numbers.map((n, i) => {
if (i % 2 === 1) return n * 2
else return n
} )
